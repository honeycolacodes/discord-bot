# 🤖 Discord Bot

A Python Discord bot with fun built-in commands.

## Commands
| Command | Description |
|---|---|
| `!hello` | Bot greets you by name |
| `!flip` | Flips a coin |
| `!roll [sides]` | Rolls a dice (default 6 sides) |
| `!joke` | Sends a random programming joke |
| `!helpme` | Lists all commands |

## How to use
1. Clone this repo
2. Install dependencies:
   pip install discord.py
3. Create a bot at discord.com/developers/applications
4. Replace `YOUR_BOT_TOKEN_HERE` in `bot.py` with your token
5. Run: `python bot.py`

## Tech used
- Python 3
- `discord.py` — Discord API wrapper
